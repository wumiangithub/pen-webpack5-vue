# webpack5 for vue 
```bash

如果你觉得@vue/cli脚手架4和5都太慢的话，可以尝试使用本项目

If you think @Vue/cli scaffolding 4 and 5 are too slow, you can try this project


```

## 📦 Start
```bash
clone this project

cd project

npm install

npm start

http://localhost:2022
```

## 🖥 分析代码体积构成
```bash

yarn analyzer

```

## 🔨 Usage
```bash

yarn prettier   格式化你的代码

yarn build  构建你的生产代码

```
